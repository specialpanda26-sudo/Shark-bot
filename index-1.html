Features

Everything packed into
one WhatsApp bot.

🤖

AI Chat via Llama-3

Ask anything with /ask and get a response from Meta's Llama-3 8B model running through the Groq API — fast inference, no hallucination-prone tiny models. Knowledge, explanations, code, summaries — right inside WhatsApp.

Powered by Groq
🎬

Video & Audio Downloads

Download videos from YouTube, Instagram, TikTok and more with /download_video. Extract MP3 audio with /download_song. Powered by yt-dlp and ffmpeg.

😮

Auto Reactions

The bot reads the sentiment of every incoming message and reacts with an appropriate emoji automatically — no command needed.

🔒

Anti-Ban Protection

Built-in fake typing indicators and rate-aware behaviour make the bot behave like a real user, reducing the risk of WhatsApp account suspension.

🗃️

Message Recovery

Every message is logged to SQLite. Use /recover to pull back the last 5 messages from any contact — deleted messages included.

⚡

Async Python Backend

The entire command processor runs on Quart — a fully async Python web framework — so downloads, AI calls, and DB writes never block each other.

01

Message arrives on WhatsApp

The Node.js client (client_bridge.js) connects to WhatsApp Web and listens for incoming messages in real time.

02

Forwarded to Python API

Every message is POSTed to the Quart backend at /webhook. The backend logs it, registers new users, and parses the command.

03

Response dispatched

Depending on the command, the backend calls Groq's API, runs yt-dlp as a subprocess, or queries SQLite — then returns the result to WhatsApp.

Deploy

Run it yourself.

One Docker container. One env variable. Free hosting on Render.

01

Clone the repo

git clone https://github.com/specialpanda26-sudo/Shark-bot.git
cd Shark-bot
02

Set your API key

GROQ_API_KEY=your_groq_api_key_here

Get a free key at console.groq.com

03

Run with Docker

docker build -t henry-tech .
docker run -e GROQ_API_KEY=$GROQ_API_KEY henry-tech
☁️

Or deploy free to Render

Push to GitHub, connect your repo on Render.com, set GROQ_API_KEY as an environment variable, and auto-deploy kicks in on every push.
